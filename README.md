# Modelos - Capstone: [A. C. Camargo] Máquina para DNA Combing

Arquivos para os modelos criados para o projeto Capstone, realizado em parceria com o **Hospital A.C.Camargo Cancer Center**.

## Estrutura do projeto

- `desenho-tecnico/`: arquivos de desenho técnico.
- `modelos/`: arquivos de modelagem 3D.
- `pcb/`: esquemática da PCB.

## WBS e Análise de Riscos

- [WBS](WBS.html): estrutura analítica do projeto (Work Breakdown Structure).
- [Análise de Riscos](Análise%20de%20Riscos.xlsx): planilha com identificação e avaliação de riscos do projeto.
